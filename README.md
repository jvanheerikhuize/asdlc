# asdlc-rebuild

Clean-slate redesign of a governed, agent- and vendor-agnostic Agentic
Software Development Lifecycle (ASDLC) framework, for a regulated
(DORA / EU AI Act) context, hosted on GitHub.

Start with [PURPOSE.md](PURPOSE.md) — it holds the current state of the work,
the researched design positions, and what comes next. The original design
brief lives in [docs/](docs/).
